Archival project: UMD Movies

July 19, 2020 — Kiiwii

Hello everyone! I recently got a PSP for the first time and have been loving it so far! I heard that there were movies for the PSP so I thought it'd be neat to get some archived versions of it but couldn't seem to find any! So I decided to start my own archival project for them! I am going to start collecting as many UMD movies as I can and when I get a new addition to my collection I'll back it up and store it online (privatley though, don't want sony to come knocking down my door lol). I will however have a table of all the movies I have backed up with the checksum and size of the .iso file when ripped! I'm really hoping that one day I can get a community going for this archive project where users can add to the backup and we can have a full table of all the movies and TV shows! I've ordered my first to "movies" (really they're TV shows) and they are Jackass Vol 2 & 3! They were the cheapest on ebay when I bought them so that's why :p


I'll be sure to update about this when I can!
